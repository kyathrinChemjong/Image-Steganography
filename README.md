# 🔐 Image Steganography

A web-based application that hides secret messages inside images using AES encryption. Built as a **Minor Project** for [Lalitpur Engineering College].

## ✨ Features

- ✅ Hide secret text inside images
- ✅ Extract hidden messages from encoded images
- ✅ AES encryption for security
- ✅ User-friendly web interface
- ✅ Machine learning model for accuracy prediction

## 🛠️ Technologies Used

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript, Bootstrap 4
- **Encryption:** PyCryptodome (AES)
- **ML Model:** Scikit-learn
- **Database:** SQLAlchemy
